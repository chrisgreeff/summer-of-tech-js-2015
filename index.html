So like any language there are a number of different types in JavaScript. You're probably familiar with most of these, except for maybe NaN and undefined. I won't go through all of these in detail, but just to give you a quick overview if you aren't already familiar

Loosely Typed


var myNum = 32;

var myStr = 'Puppies!';

var no = false;
Before I move onto move details about the types, I should mention that JavaScript is a loosely typed language. So unlike Java or C, you don't have to be explicit in the type of variable you are declaring. You can just use the keyword `var`, and JavaScript will work out what you're talking about.

Objects

To keep things simple, you can think of objects as basic maps of key value pairs. So for example defining myObj, with a property called. Interacting with the property is done through the dot notation

NaN

As a quick intro to these concepts, NaN is a mathematical concept in which rather than throwing an error when an impossible mathematical operation is performed, the result NaN is returned. So for example 2/'derp' will give the result NaN.

Undefined

Undefined is a little simpler to understand. It's basically any declared variable or object property that has not been assigned a value by default gets the value undefined.

Functions

Like other language JavaScript also has the concept of functions, which you can define using the `function` keyword, and invoke with parameters using brackets.
